Google Drive Setup Guide
  1. Create a Google account (or use an existing one)
  2. Go to console.cloud.google.com → New Project → name it AttendanceTracker
  3. Enable Google Drive API and Google Sheets API
  4. Create credentials → OAuth 2.0 Client ID (Web application type)
  5. Add the Authorized JS origin and Redirect URI shown below
  6. Create credentials → API Key, restrict to Drive + Sheets APIs
  7. Paste both into the fields below and click Connect

📋 Attendance Policy Reference

Point System

Points accumulate over a rolling 12-month period. Reaching the mid threshold triggers a warning; reaching the high threshold triggers discipline.

Occurrence Types

TypeDescription
OccurrenceUnexcused absence or late arrival
NCNSNo call no show — most severe
Future RequestPre-approved upcoming absence
Past RequestRetroactive time off request

Flags

  • Cluster — Part of a recurring pattern
  • 30-Day Give Back — Points being returned per policy
  • Policy Drop Off Pending — Eligible for removal after clean period
